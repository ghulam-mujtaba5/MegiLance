# 3. Visual Identity

## 3.1 Logo
- Symbol: Minimalist logo with an abstract blockchain link, person/freelancer icon, and optional AI node symbol
- Style: Flat, modern, scalable
- Color versions: Primary, Monochrome, White-on-dark
